## Описание
Отдельный остров из пяти частей бывшего острова Ирлин.

## Ключевые объекты
- **Академия Южного Келлера (руины)** — там должна быть информация о негорящих тканях для воздушного шара

## Зачем туда
- Найти сведения о негорящих тканях → для постройки воздушного шара

## Как добраться
Только через телепорт (Ленивец). Между островами нет другого способа перемещения.
